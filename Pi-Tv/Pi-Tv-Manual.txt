Pi-TV — Raspberry Pi setup manual
=================================

Quick checklist: HAT + aerial → dtoverlay=tvhat → reboot → install → log out/in → pitv


1. Hardware
-----------

  1. Fit the official Raspberry Pi TV HAT on the Pi.
  2. Connect a DVB-T/T2 aerial to the HAT.
  3. Connect a keyboard (or remote that sends arrow keys + Enter).
  4. Use a display with Raspberry Pi OS with desktop (Pi-TV needs the desktop
     for the menu window and video).


2. Get Pi-TV onto the Pi
------------------------

  Option A — Git (only if GitHub has the latest files):

    cd ~
    git clone https://github.com/JackTheHack247/Pi-Tv.git
    cd Pi-Tv

  Option B — Copy from your PC (recommended if GitHub is not fully up to date):

    Copy the whole Pi-Tv folder to the Pi (USB, SCP, etc.), then:

    cd ~/Pi-Tv

    (or wherever you copied it)


3. Enable the TV HAT
--------------------

  Edit the boot config:

    sudo nano /boot/firmware/config.txt

  On older OS, use /boot/config.txt instead.

  Add this line (if it is not already there):

    dtoverlay=tvhat

  Save, then reboot:

    sudo reboot

  After reboot, check the HAT is seen:

    ls /dev/dvb/

  You should see something like adapter0.


4. Install Pi-TV
----------------

    cd ~/Pi-Tv
    chmod +x scripts/install-pi.sh pitv-run
    ./scripts/install-pi.sh

  That installs:

    - mpv, DVB tools, scan tables, Python Tk, etc.
    - Pi-TV to ~/pitv (by default)
    - a pitv command in ~/.local/bin


5. Log out and back in
----------------------

  If the installer added you to the video or input groups, log out and log
  back in (or reboot). Without that, the HAT or keyboard may not work properly
  while watching TV.


6. Run Pi-TV
------------

    pitv

  If pitv is not found:

    export PATH="$HOME/.local/bin:$PATH"
    pitv

  Or:

    cd ~/pitv
    ./pitv-run

  The menu opens full screen. For a smaller window while testing:

    PITV_WINDOWED=1 pitv


7. First time using it
----------------------

  1. Pick your country/region (↑/↓, Enter) — it scans your aerial (can take
     a minute).
  2. It loads only channels your aerial actually receives (no internet needed).
  3. TV starts in the same window on channel 1.
  4. ↑/↓ changes channel while watching.
  5. A programme info box (channel, title, description) appears top-right when
     you change channel, then fades after a few seconds.
  6. Enter opens the country list. Esc closes Pi-TV and returns to the desktop.


8. Controls
-----------

  Menu                    | Key              | Action
  ------------------------|------------------|----------------------------------
  Menu                    | ↑ / ↓            | Move selection
  Menu                    | Enter            | Select country or channel
  Menu                    | Esc              | Quit (back to desktop)
  While scanning          | Enter on ← Close | Cancel scan
  While watching          | Enter            | Country list (← Back to TV selected)
  Country list            | Enter on ← Back to TV | Same channel as before
  Country list            | ↓ + country + Enter   | Rescan/load → channel 1
  Channel list            | Change region    | Go back to country list
  While watching          | ↑ / ↓            | Change channel
  While watching          | Esc              | Quit (back to desktop)

  On a TV remote, Back/Exit usually works like Esc.


9. Colours (optional)
---------------------

  Edit menu and overlay colours in either place:

    ~/pitv/pitv/ui/window.py   (DEFAULT_THEME block)

  or

    ~/.config/pitv/config.json

  Example config.json:

    {
      "theme": {
        "bg": "#ffffff",
        "text": "#111111",
        "sidebar_bg": "#f3f3f3",
        "sidebar_text": "#222222",
        "selected_bg": "#eeeeee",
        "muted": "#666666",
        "status_bg": "#fff8e1",
        "status_text": "#664d00",
        "border": "#dddddd",
        "overlay_bg": "#111111d9",
        "overlay_border": "#666666",
        "overlay_channel": "#ffffff",
        "overlay_program": "#ffffff",
        "overlay_description": "#cccccc",
        "overlay_muted": "#888888",
        "overlay_radius": 10
      }
    }

  Menu keys (bg, text, sidebar_bg, …) style the list window.

  Overlay keys (overlay_*) style the programme box while watching TV:
    overlay_channel     — channel name text
    overlay_program     — programme title text
    overlay_description — description text
    overlay_muted       — "Channel 3 / 47" and key hints
    overlay_bg          — box background (8-digit hex can include transparency)
    overlay_border      — box border
    overlay_radius      — corner roundness (pixels)

  Restart pitv after changes.


10. Rescan channels later
-------------------------

  Pick a country again from the menu, or run:

    python3 -m pitv.dvb_scan


11. If something goes wrong
---------------------------

  Problem                  | Try
  -------------------------|--------------------------------------------------
  "No TV HAT found"        | Check dtoverlay=tvhat, reboot, ls /dev/dvb/
  No channels after scan   | Check aerial, try rescan, improve signal
  Black video              | PITV_VO=x11 pitv
  Keyboard dead while TV   | Log out/in after install (input group)
  pitv not found           | export PATH="$HOME/.local/bin:$PATH"
